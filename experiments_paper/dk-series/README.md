Folder `results` contains the average distance between *0k*-, *1k*- and *2k*-random graphs, generated from the six source networks discussed in the manuscript.  
These results can be reproduced with `analysis.py`.  The random graphs used in the paper are stored in `data.zip` (must be extracted before running `analysis.py`!).  

Fig. 3 in the manuscript can be reproduced by running `plot.py`.  

*0k*-random graphs have been generated using `generate_dk0.py`. For *1k*- and *2k*-random graphs, we have used the [RandNetGen](https://github.com/polcolomer/RandNetGen) software, available on GitHub.
